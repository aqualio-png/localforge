À quoi ça sert ?

LocalForge est votre atelier personnel pour créer et publier des applications web, sans payer d'abonnement à des services comme Netlify ou Emergent.sh.

Vous décrivez ce que vous voulez en français, Claude génère l'application, et vous pouvez la publier gratuitement sur Internet via GitHub Pages.

Partie 1 — Configuration initiale

À faire une seule fois. Ensuite tout fonctionne automatiquement.

A. Obtenir votre clé Claude (IA)

1

Créer un compte Anthropic

Allez sur console.anthropic.com et créez un compte gratuit.

2

Générer une clé API

Une fois connecté : menu gauche → API Keys → Create Key. Copiez la clé sk-ant-api03-...

3

Saisir la clé dans LocalForge

Dans LocalForge → ⚙ Config. (en bas sur mobile) → onglet ⬡ Claude API → collez la clé → Enregistrer.

⚠
La génération d'apps avec Claude est payante à l'usage (quelques centimes par app). Ajoutez un peu de crédit sur console.anthropic.com → Billing.

B. Configurer GitHub (publication gratuite)

4

Créer un compte GitHub

Allez sur github.com et créez un compte si ce n'est pas déjà fait. C'est gratuit.

5

Créer un token d'accès

Sur github.com : cliquez votre photo en haut à droite → Settings
→ tout en bas à gauche : Developer settings
→ Personal access tokens → Tokens (classic)
→ Generate new token (classic)
→ Donnez un nom (ex: LocalForge), cochez la case repo
→ Cliquez Generate token en bas
→ Copiez le code ghp_... — il n'apparaît qu'une fois !

6

Saisir dans LocalForge

Dans LocalForge → ⚙ Config. → onglet 🐙 GitHub → remplissez votre username et le token → Enregistrer.

💡
Cette configuration n'est faite qu'une seule fois. LocalForge mémorise tout dans votre navigateur.
Partie 2 — Créer une application
1

Cliquer sur "+ Nouveau projet"

En bas de l'écran sur mobile, ou en haut sur ordinateur.

2

Choisir une icône et un nom

Sélectionnez une icône parmi celles proposées. Donnez un nom à votre projet (ex: Suivi de budget).

3

Décrire votre application

Dans le champ "Prompt", expliquez en français ce que vous voulez :
Ex : "Un tableau de bord avec un graphique de mes dépenses mensuelles et un formulaire pour ajouter des entrées."
Plus vous êtes précis, meilleur sera le résultat.

4

Cliquer "⬡ Générer avec Claude"

Claude génère votre application en 15 à 60 secondes. Vous verrez le code apparaître progressivement.

5

Votre app est prête !

Elle s'ouvre automatiquement dans LocalForge. Vous pouvez la prévisualiser, la modifier, ou la publier.

Partie 3 — Publier sur Internet
1

Aller dans l'onglet "⚡ Actions"

Quand votre projet est ouvert, appuyez sur l'onglet ⚡ Actions en haut.

2

Cliquer "🐙 Déployer sur GitHub Pages"

LocalForge crée automatiquement un dépôt sur GitHub et y publie votre app.

3

Attendre ~60 secondes

GitHub prépare votre site. Une URL violette apparaît dans l'app :
votre-username.github.io/nom-projet

4

Partager le lien !

Ce lien est public, permanent, gratuit. Envoyez-le à qui vous voulez.

Questions fréquentes

Est-ce vraiment gratuit ?

GitHub Pages est 100% gratuit. Seule la génération IA (Claude) est payante à l'usage — environ 1 à 3 centimes par application générée selon sa complexité.

Mes données sont-elles sécurisées ?

Vos clés API sont stockées uniquement dans votre navigateur, jamais envoyées ailleurs. Vos projets sont aussi sauvegardés localement dans votre navigateur.

L'app générée ne me convient pas ?

Allez dans l'onglet ⟳ Modifier, changez ou précisez votre description, et relancez la génération. Vous pouvez recommencer autant de fois que vous voulez.

Je ferme le navigateur, je perds tout ?

Non. Vos projets sont sauvegardés automatiquement dans le navigateur. Ils seront là à votre retour, sauf si vous videz les données du navigateur.
